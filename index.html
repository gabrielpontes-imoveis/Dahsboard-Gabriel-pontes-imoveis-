⚙️ Configurar APIs

Insira suas credenciais para conectar às plataformas e exibir dados reais. As chaves são salvas localmente no seu navegador.

▶️ YouTube Data API

Obtenha em console.developers.google.com → YouTube Data API v3

Encontre em youtube.com → Seu Canal → Sobre → Compartilhar → Copiar ID

📸 Instagram Graph API

Obtenha via Facebook Developers → Instagram Basic Display API

Retornado junto com o access token na API do Facebook

🎵 TikTok (Dados Manuais)

A API do TikTok é restrita a contas business. Insira manualmente seus dados do TikTok Analytics.
